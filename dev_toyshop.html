overview

In this assignment,
we needed to make an app displaying the list
of modules and upon pressing it, it links to a new page
with information to that module

process

Adding the text was easy,
the part we had to code in this was that when
the user clicks on the name of that module
it would lead to a new page.

Additionally, we only made one single page.
So we learnt something called Intent.
Essentially, the intent stores the information and would
replace the placeholders if that module is clicked

takeaways

I learnt how to use Intent so that
I can reduce the effort and the number of pages
I needed to make in the future.
