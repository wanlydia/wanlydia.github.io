overview

In this project, we were tasked to create a portfolio on a mobile phone. There were three important criteria in this: Figure out my personal branding, feature my best works, and demonstrate my skill and understanding in UI/UX design.

process


From my previous experiences, I found moodboard to be especially useful at setting the theme of the project. So I used it to figure out my personal branding. I wanted it to feature my two favourite colours - dark blue and yellow. I also wanted to add a 'whimsical' feel to it, so there were stars, skulls and cards. And as an icing to the cake, the font I used is called 'Chippewa Falls' which suits the theme.

I wanted to try something unique with the home page, so I created a bedroom that feature objects that links to the other pages. A mirror for 'About Me', a bookshelf for 'Portfolio', and scattered letters for 'Contact Me'.

There were no hover features as it was a mobile app. I applied an auto-animate feature to the app that occurs when the user submits the contact form. It sends the user to a loading screen and then to confirmation page. The hamburger navigation menu was hidden unless the user decides to interact with it. The hamburger icon would then transform to an 'X' upon clicking it.

takeaways

From this activity, I was able to put myself into the perspective of a mobile app developer. Because the format was mobile, it encourages me to think of ways to make this site usable, especially with it's small space.
