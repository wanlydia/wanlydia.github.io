to do list

overview

In my Android lesson,
we were tasked to create a to do list app.

process

I had to add the text and inputs required.
A new concept we learnt was an array list.
This allowed us to add the tasks to the list
as well as remove it from the list

As for the design,
I think giving it a bright, modern look was suitable
as that is the trend I see in current 'To Do List' apps.

takeaways

From this lesson, I learnt to use an array list
for my apps should I decide to show lists.
