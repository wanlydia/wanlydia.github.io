magazine

overview

We were tasked to create a magazine page.
We have the images and the text already,
so we needed to assemble it.

process

In this lesson, we learnt something called grids.
It is basically a placeholder of sorts.
We create small boxes across the page and these boxes
are how we visualise a piece of text of image is placed.

First, I added a padding from the edges
so that it is comfortable when reading.

There were two parts to this article.
I assigned the first page as where the introductions are,
splitting the paragraphs into two.
On the second page, I wanted to do this
alternate-the-pictures-position thing as it moves
from one product to another.

takeaways

From this assignment,
I learnt how to use grids on Adobe.
