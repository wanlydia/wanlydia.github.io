This assignment required us to create a

process

One of the first things we needed to do was to create
a logo in an svg format. This meant that we needed
to use Adobe Illustrator to construct it.
The idea behind this logo is that I took the initials
of the website name. But because I wanted to show that it
is a travel website, I turned one part of the 'X'
into an airplane.

Afterwards, I created a quick lo-fi to figure out
where I wanted to put each content.

When it comes to coding the website, we needed
to add a video to the website and it would play continuously
in the background WITH text and button on it.
In my mind, that sounded cool but doing it code,
it is a whole different story.
I still have formatting problems but the video is playing
so I am happy with the product

takeaways

I learnt to add a video loop to the website
as well as converting an image to an SVG.
