subpar-market

overview

In one of my HTML lessons,
we're introduced to a CSS Framework called Bootstrap

We were to apply it to make a supermarket website
with three products from two different categories.

process

The idea behind the name Subpar-Market
was that I knew that the final product would not pretty
but I enjoy a good wordplay.

For this assignment,
I needed to set up a top navigation bar with
one dropdown and a contact page.
Additionally, I had to program Bootstrap cards
for three products that would have a pop-up upon clicking.

takeaways

From this project, I learnt how to use Bootstrap
and how it serves as a helping tool when creating a website.
However, I would have also made improvements such as
ensuring that the product images are all of the same size so that
it would look a lot more uniform.
